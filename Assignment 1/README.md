# NTHU Natural Language Processing (2025) — Assignment 1

This repository contains my implementation and report for **Assignment 1: Word Embeddings** from the NTHU Natural Language Processing course (2025).

## Contents
- `src/`  
  Source code for preprocessing the Google analogy dataset, evaluating pre-trained embeddings, and training custom word embeddings.
- `reports/`  
  Public version of the report.

## Highlights
- Parsed the Google Analogy dataset into a structured format with semantic vs. syntactic categories.
- Evaluated pre-trained word embeddings on analogy tasks.
- Trained custom Word2Vec embeddings on a sampled Wikipedia corpus and compared performance.
- Visualized word relationships using t-SNE.
